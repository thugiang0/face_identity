# Face Identity

## Detection
- MTCNN

## Recognition
- Insightface
- Facenet

## How to use
- Clone
    ```
    git clone https://github.com/thugiang0/face_identity.git
    ```

- Install
     ```
    pip -r install requirements.txt
    ```

- Inference
    + Face Recognition with Insightface
    ```
    python test.py
    ```

    + Face Recognition with Facenet
    ```
    python test_facenet.py
    ```
